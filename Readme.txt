# Company Profile Website - PT Manunggal Teknologi Indonesia

Website company profile modern untuk PT Manunggal Teknologi Indonesia yang dibuat menggunakan template BootstrapMade dan telah dimodifikasi serta dikembangkan kembali sesuai kebutuhan perusahaan.

## Overview

Project ini merupakan website company profile yang digunakan untuk menampilkan informasi perusahaan secara profesional di internet. Website dirancang dengan tampilan modern, responsive, dan user friendly agar dapat diakses dengan baik di desktop maupun mobile device.

## Features

- Responsive Design
- Modern UI/UX
- Company Information Section
- Services Section
- Portfolio Section
- Contact Section
- SEO Friendly Structure
- Smooth Navigation
- Bootstrap Based Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- BootstrapMade Template
